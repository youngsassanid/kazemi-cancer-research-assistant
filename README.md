# Kazemi Cancer Research Assistant

The **Kazemi Cancer Research Assistant** is a web-based tool designed to support cancer research and data visualization.  
It provides an interactive dashboard for analyzing cancer-related datasets, visualizing statistics, and assisting medical research efforts with user-friendly graphs and insights.

---

## Features

- 📊 **Interactive Graphs** using Plotly.js for real-time data visualization  
- 🔍 **Data Exploration Tools** for cancer research datasets  
- 🎨 **Modern UI/UX** with custom CSS styling  
- 📱 **Responsive Design** optimized for both desktop and mobile  
- 🧩 **Modular Code Structure** for easy extension and customization  

---

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript  
- **Visualization:** [Plotly.js](https://plotly.com/javascript/)  
- **Deployment:** Works as a static web app (host on GitHub Pages, Vercel, Netlify, etc.)

## Setup & Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/kazemi-cancer-research-assistant.git
````

2. Open the project folder:

   ```bash
   cd kazemi-cancer-research-assistant
   ```

3. Open **cancer_research_assistant.html** in your browser:

   ```bash
   open cancer_research_assistant.html   # Mac
   start cancer_research_assistant.html  # Windows
   xdg-open cancer_research_assistant.html  # Linux
   ```

No server setup required — it’s a fully client-side app.
